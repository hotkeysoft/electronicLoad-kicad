# Electronic Load
Rev A July 2021

***Work in progress, do not use***

- USB Powered (USB-B Connector)
- 0-24V DC Input
- Adjustable current range: 0-4A
- I2C OLED 0.96" Display
- Rotary Encoder w/switch
- Temperature Sensors w/hardware shutdown
- 5V Cooling fan (PWM Speed Control)
